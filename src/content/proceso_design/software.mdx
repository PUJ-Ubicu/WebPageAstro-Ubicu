Los incentivos respiratorios actuales presentan ciertas limitaciones, ya que su efectividad depende en gran medida del seguimiento y la monitorización constante, lo que puede dificultar su adherencia por parte de los pacientes. Para abordar este desafío, se busca disminuir dicha limitación con gamificación, al crear intervenciones agradables y garantizar compromiso a largo plazo. Para este fin, se desarrolló un sistema web de gestión remota para fisioterapeutas y una aplicación móvil gamificada para pacientes que apoye fisioterapia respiratoria durante la ejecución de ejercicio de reexpansión pulmonar, en este caso, el uso de un incentivo respiratorio, integrados a un inspirómetro digital. El objetivo es orientar y hacer seguimiento del estado de evolución del paciente a distancia, al tiempo que se incentiva a este con un juego basado en prescripciones. Para el desarrollo del sistema se siguió la metodología de desarrollo de software XP (Extreme Programming), la cual es una metodología de desarrollo ágil.
