Se diseñó, se modeló y se simuló el sistema de medida de flujo con tubo Venturi a partir de recomendaciones estandarizadas de instrumentación. Se diseñó un sistema para motivación del paciente basado en un videojuego con una nutria para manejarse en un dispositivo móvil. Se configuró la comunicación entre el dispositivo de medida y el dispoisitvo móvil por Bluetooth, como también del dispositivo móvil a un programa de administración y control de pacientes en la nube.
